# Kentucky War Project - Website Information

## Project Overview

**Kentucky War** is a military-themed RolePlay server website for GTA V, featuring:

- Military-focused gameplay
- Multiple combat factions
- Tactical operations and large-scale battles
- Rank progression system
- Military equipment and vehicles

## Design Philosophy

### Visual Identity

- **Color Scheme**: Dark military green palette
- **Typography**: Bold, uppercase headers with military aesthetic
- **Imagery**: Combat-focused, tactical operations
- **Atmosphere**: Serious, professional military environment

### Key Design Elements

1. **Dark Theme**: Creates immersive military atmosphere
2. **Green Accents**: Military camouflage colors (#2d5016, #4a6741)
3. **Combat Iconography**: Military symbols and badges
4. **Tactical Layout**: Clean, organized structure

## Content Structure

### Main Sections

1. **Hero Section**
   - Project name and tagline
   - Online player counters
   - Call-to-action buttons

2. **About Section**
   - Project description
   - Key statistics
   - Military focus explanation

3. **Factions Section**
   - Three military factions:
     - Blue Forces (United Forces)
     - Red Forces (Liberation Army)
     - Green Forces (Special Operations)
   - Unique characteristics per faction

4. **Servers Section**
   - Active server list
   - Player counts
   - Server status indicators

5. **Features Section**
   - Military vehicles
   - Rank system
   - Tactical operations
   - Military economy
   - Tournaments and events

6. **Donation Section**
   - Three tier system:
     - Recruit (199₽)
     - Sergeant (499₽)
     - Commander (999₽)

7. **Getting Started**
   - Step-by-step guide
   - Installation instructions
   - Launcher download

8. **FAQ Section**
   - Common questions
   - Collapsible answers

## Technical Features

### Responsive Design

- Mobile-first approach
- Breakpoints:
  - Desktop: 1024px+
  - Tablet: 768px - 1023px
  - Mobile: < 768px

### Interactive Elements

- Smooth scrolling
- Animated counters
- FAQ accordion
- Server IP copy function
- Hover effects and transitions

### Performance

- Pure HTML/CSS/JS (no frameworks)
- Optimized animations
- Lazy loading compatible
- Fast load times

## Differences from Kentucky V

### Theme Changes

| Aspect | Kentucky V | Kentucky War |
|--------|-----------|-------------|
| Color Scheme | Vibrant, modern | Dark military green |
| Focus | RolePlay variety | Combat operations |
| Factions | Various roles | Military units |
| Atmosphere | Urban lifestyle | Tactical warfare |
| Design | Modern gaming | Military tactical |

### Content Adaptations

1. **Military Terminology**
   - "Бойцы" instead of "Игроки"
   - "Операции" instead of "События"
   - "Фракции" focus on military units

2. **Feature Focus**
   - Vehicles → Military vehicles
   - Business → Military economy
   - Events → Tactical operations

3. **Visual Style**
   - Darker backgrounds
   - Camouflage-inspired colors
   - Combat-focused imagery
   - Tactical iconography
